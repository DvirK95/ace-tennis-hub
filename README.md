# 🎾 Ace Tennis Hub

A complete full-stack web application for managing a tennis club. This project features a modern React frontend and a robust Express backend, utilizing a unified Monorepo structure.

## 🏗️ Project Architecture (Monorepo)

This repository is structured as a Monorepo containing both the client and server applications:

- **/frontend:** The user interface, built with React and Vite.
- **/backend:** The RESTful API server, built with Node.js, Express, and Prisma.

## 💻 Tech Stack

### Frontend

- **Framework:** React (via Vite)
- **Language:** TypeScript
- **State Management:** Zustand & React Query
- **Forms & Validation:** react-hook-form + Zod
- **Tables:** TanStack Table

### Backend

- **Runtime:** Node.js
- **Framework:** Express.js
- **Language:** TypeScript
- **ORM:** Prisma (v7 with Driver Adapters)
- **Database:** PostgreSQL (Hosted on Supabase)
- **Validation:** Zod
- **API Documentation:** Swagger UI

## 🚀 Getting Started

Follow these instructions to set up the project locally on your machine.

### Prerequisites

- Node.js (v18 or higher)
- A Supabase account (for the PostgreSQL database)

```bash
git clone [https://github.com/YOUR-USERNAME/Ace-tennis-hub-mern.git](https://github.com/YOUR-USERNAME/Ace-tennis-hub-mern.git)
cd Ace-tennis-hub-mern
```
